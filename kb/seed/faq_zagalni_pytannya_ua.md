# Загальні питання

## Що таке Acme Cloud Suite?

Acme Cloud Suite — це сервіс для управління проєктами та задачами команд від компанії Acme Inc.
Він допомагає командам планувати роботу, відстежувати завдання на дошках і в списках,
співпрацювати над проєктами та підключати інші інструменти через REST API. Acme Cloud Suite
працює у браузері, а також має десктоп-застосунок, який тримає локальний кеш синхронізованим із
хмарним акаунтом.

## Для кого призначений Acme Cloud Suite?

Продукт підходить командам будь-якого розміру:

- **Невеликі команди та фрілансери**, яким потрібен простий спосіб відстежувати завдання,
  можуть почати з тарифу Free.
- **Команди, що ростуть**, зазвичай переходять на Starter або Pro, коли потрібно більше
  користувачів і місця для зберігання даних.
- **Великі організації** з вимогами до відповідності стандартам, єдиного входу (SSO) та
  розширеної підтримки зазвичай обирають тариф Enterprise.

## Які тарифні плани доступні?

Acme Cloud Suite пропонує чотири тарифи з помісячною оплатою за користувача:

| Тариф | Ціна | Користувачі | Сховище |
|---|---|---|---|
| Free | $0 | До 3 користувачів | 1GB |
| Starter | $9/користувач/місяць | До 20 користувачів | 20GB |
| Pro | $19/користувач/місяць | Необмежена кількість користувачів | 200GB |
| Enterprise | Індивідуальна ціна через відділ продажів | Необмежена кількість користувачів | Необмежене сховище |

Для тарифів Starter і Pro доступна річна оплата: ви платите за 10 місяців, а отримуєте 12
місяців обслуговування (знижка приблизно 17% порівняно з помісячною оплатою). Детальніше — у
документі "Тарифи та оплата".

## Чи є в Acme Cloud Suite API?

Так. Acme Cloud Suite надає REST API за адресою `https://api.acmecloudsuite.example.com/v1` для
інтеграцій, автоматизацій та власних інструментів. Автентифікація відбувається за допомогою
API-ключа, який передається в заголовку `Authorization: Bearer <key>`. Ліміти запитів залежать
від тарифу — дивіться документ "Посібник з API".

## Яка підтримка мені доступна?

Час відповіді підтримки залежить від тарифу:

- **Free та Starter**: відповідь електронною поштою протягом 48 годин.
- **Pro**: пріоритетна підтримка з відповіддю протягом 24 годин.
- **Enterprise**: відповідь протягом 4 годин, персональний менеджер акаунта та підтримка по
  телефону.

## Чи безпечні мої дані?

Двофакторна автентифікація (2FA) на основі TOTP (одноразові коди на основі часу) доступна на
всіх тарифах, включно з Free. Єдиний вхід (SSO) через SAML доступний лише на тарифі Enterprise.

## Що відбувається, якщо я скасую акаунт?

Після скасування дані акаунта зберігаються протягом 90 днів на випадок, якщо ви захочете
відновити акаунт. Після 90 днів дані видаляються назавжди. Детальніше — у документі "Політика
повернення коштів та скасування".

## Куди звертатись з технічними проблемами?

Якщо виникли проблеми зі входом (зокрема коди 2FA не збігаються), дивіться документ "Усунення
несправностей: проблеми зі входом". Якщо десктоп-застосунок завис на синхронізації або працює
повільно, дивіться документ "Troubleshooting: Sync & Performance".
